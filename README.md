## Overview

**Floating WhatsApp Modern UI** adalah snippet HTML–CSS–JavaScript ringan yang dirancang untuk meningkatkan konversi website melalui WhatsApp tanpa plugin berat atau dependency kompleks.

Dirancang dengan pendekatan **UX-driven micro-conversion trigger**, komponen ini memaksimalkan visual attention tanpa mengganggu pengalaman pengguna.

---

## Features

- **Modern & Premium UI**
  Gradient button + floating animation + pulse effect

- **Fully Responsive**
  Behavior berbeda untuk desktop & mobile

- **Auto Open (5s delay)**
  Attention trigger tanpa agresif

- **Desktop Interaction**
  Hover → show CTA bubble

- **Mobile Interaction**
  Click icon → toggle CTA bubble

- **Dynamic CTA Bubble**
  Blink effect + online indicator

- **Lightweight & Fast**
  Tanpa plugin, tanpa library berat

- **Fully Customizable**
  Warna, nomor, pesan, animasi bisa diubah

---

## Technical Implementation

Script ini menggunakan:

- `matchMedia()` → Deteksi perangkat sentuh
- `encodeURIComponent()` → Format pesan WhatsApp otomatis
- Conditional event handling → UX optimal desktop & mobile
- Non-blocking render → SEO-safe
- Pure HTML + CSS + Vanilla JavaScript

Tidak ada framework dependency.

## Installation

1. Copy seluruh snippet HTML.
2. Paste sebelum closing `</body>` tag.
3. Ganti:

